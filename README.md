# photo-log

Node time lapse video creator

## Install

Linux:
```sh
sudo apt-get install fswebcam
yarn
```

Windows:
```ps1
yarn
```

## Run

### Clear images

Clear all the images in the `imgs` folder

```
yarn clear
```

Confirm the prompt

### Capture images

Capture images on a schedule

Edit the code with the schdule then

```
yarn start
```
